# #01 Content Repurposing Pipeline

**Use case:** Repurpose a single video transcript into 4 platform-specific content pieces with zero overlap.

---

Create an agent team to repurpose a video transcript into content for 4 platforms.
The transcript is at /path/to/your/transcript.md.

Spawn 4 teammates:

1. **Blog Writer** — Transform the transcript into a 1,200-word blog post. Structure: hook paragraph, 3-5 subheadings with key insights, conclusion with CTA. Write in first person. Save to outputs/agent_teams_demo/content_repurposing/blog_post.md.

2. **LinkedIn Writer** — Write a LinkedIn post (150-250 words). Use a hook line, 3 key takeaways as short paragraphs, and end with a question to drive comments. No hashtag spam — max 3 relevant hashtags. Save to outputs/agent_teams_demo/content_repurposing/linkedin_post.md.

3. **Newsletter Writer** — Write an email newsletter edition (300-500 words). Subject line + preview text + body. Conversational tone, one clear CTA. Save to outputs/agent_teams_demo/content_repurposing/newsletter.md.

4. **Twitter Thread Writer** — Write a 7-10 tweet thread. Tweet 1 is the hook. Each tweet is self-contained but builds on the last. Final tweet is a CTA. Save to outputs/agent_teams_demo/content_repurposing/twitter_thread.md.

Before writing, each teammate should read the full transcript and identify the 3 most compelling insights. Have them share their chosen insights with each other to ensure no two platforms lead with the same angle — each piece should feel fresh, not repetitive.

After all 4 are done, synthesize a summary comparing the angles each teammate chose and flag any messaging inconsistencies.
