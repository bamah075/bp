# #04 Competitive Intelligence Report

**Use case:** Parallel competitor analysis with shared findings and synthesized battlecard.

---

Create an agent team to build a competitive intelligence report.

**Target product:** Claude Code — the AI-powered coding assistant CLI from Anthropic.

**Competitors to analyze:** AntiGravity, Cursor, Codex, GitHub Copilot

Spawn 4 teammates. Document the following for each platform (latest/greatest info as of 2026):

- Product overview and core features
- Pricing model and tiers
- Target audience and positioning
- Strengths (what they do better than Claude Code)
- Weaknesses (where Claude Code has an advantage)
- Recent news, funding, or product launches
- Save to outputs/agent_teams_demo/competitive_intel/[competitor]_analysis.md

Have each analyst share their top 3 findings with the group before the Synthesis Lead begins, so the Lead has context on what surprised each analyst most.

After all analysts are done, a **Synthesis Lead** builds a comprehensive intelligence report with:
- Competitive landscape overview
- Head-to-head comparison table
- Where Claude Code wins / where competitors have the edge
- Key battlecard (top 5 objections and responses)
- Strategic recommendations

Save to outputs/agent_teams_demo/competitive_intel/intelligence_report.md.
